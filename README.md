# AWS-Chime-Chat-SDK-for-Columbia-Connect-App
This is a modified AWS Chime SDK for the Columbia Connect application built for the Cloud Computing and Big Data course.

## Available Scripts

Open src/Config.js and update the configuration with the values from the stack output.

Install dependencies and start the demo in the apps/chat directory.

```
npm install
npm start
```

Open http://localhost:9000 in your browser.
